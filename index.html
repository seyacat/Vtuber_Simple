How to use:

  • Select your microphone from the dropdown menu (optional)
  • The microphone starts automatically after loading
  • Speak vowels (A, E, I, O, U) to see real-time vowel detection
  • The system uses a machine learning model trained with Teachable Machine
  • The visualization shows amplitude over time (time-domain)
  • Volume level shows the current audio intensity
  • Vowel detection shows which vowel is being spoken
  • Confidence shows how sure the system is about the detection
  • Click "Stop Microphone" to end the audio capture
  • Click "Start Microphone" to restart if needed
  • Your browser will ask for microphone permission - please allow it
  • If your microphone doesn't appear, click "Start Microphone" to refresh the device list
  • The microphone continues recording even when the tab is in the background
  • The app remembers your last selected microphone for future sessions

Built with Web Audio API & Canvas | Works in modern browsers with microphone support
